# 术语表

**判别式推荐（Discriminative）**  
对给定用户–物品对或候选集合打分，不直接建模物品联合分布。典型：双塔、DeepFM、DIN。

**生成式推荐（Generative）**  
显式或隐式建模数据生成分布，可输出离散 token 序列、语义 ID 或自然语言决策。常与自回归、扩散等生成模型关联。

**序列推荐（Sequential Recommendation）**  
以用户历史交互序列为输入，预测下一物品或下一篮子。

**语义 ID / Semantic ID**  
将物品映射为短离散 token 序列（如 RQ-VAE 码本索引），便于生成式模型在压缩空间检索。

**多阶段漏斗**  
召回 → 粗排 → 精排 → 重排 等级联；每阶段候选规模递减、模型复杂度可递增。

**NDCG / Recall / HR**  
常见离线排序指标；具体定义与截断位置 k 强相关，跨论文比较需对齐协议。

**Beam Search**  
自回归解码时保留多条高分前缀以兼顾质量与多样性的近似搜索策略。

**RLHF / 对齐**  
人类反馈强化学习等对齐技术；在推荐场景中可用于约束生成式话术与策略。
